# M3U8 批量下载器 (Rust 版)

基于 Rust 重写的批量 M3U8 下载工具，利用 `N_m3u8DL-CLI` 作为下载引擎，支持从 CSV 文件读取任务列表，并发执行下载，自动记录进度、删除完成条目，并生成错误日志。

---

## 🚀 功能特性

- **多任务并发下载** – 默认最多同时 5 个任务，可配置。
- **CSV 任务管理** – 从 `DownloadList.csv` 读取标题和 URL，下载成功后自动删除该行。
- **实时进度显示** – 每个任务独立显示下载进度（`已下载/总数 (百分比)`）。
- **错误日志记录** – 下载失败或文件缺失时，在 `Logs/` 目录生成含时间戳的详细日志。
- **自动清理** – 下载成功后立即删除 CSV 对应行，避免重复下载。
- **线程安全** – 使用互斥锁保护共享资源（CSV、终端输出），避免并发冲突。

---

## 🛠️ 系统要求

- **操作系统**：Windows（当前编译为 `.exe`，其他平台可自行编译）
- **依赖环境**：
    - 本项目依赖 **N_m3u8DL-CLI** (建议 v3.0.2 或更高版本)。
    - 请确保 `N_m3u8DL-CLI` 已正确配置在系统路径中，或放置在指定目录。

项目根目录/
├── N_m3u8DL_CLI/
│   └── N_m3u8DL-CLI_v3.0.2.exe # 下载引擎 (需自行下载放入)

## 📖 使用方法

### 1. 准备任务列表

在 `DownloadList.csv` 中按行编写任务，格式为：`标题,URL`

例如：
```csv
视频1,https://example.com/video1.m3u8
视频2,https://example.com/video2.m3u8

**注意**：

- 不要包含表头。
- 每行必须包含且仅包含一个逗号。
- 标题会作为下载文件名（无需加扩展名）。
- (建议搭配 VSCode 插件 **Edit CSV** 使用)

```
### 2. 运行程序

双击 `m3u8_downloader.exe`，或在命令行中执行：

```cmd
m3u8_downloader.exe
```

程序会自动读取 `DownloadList.csv`，开始并发下载。每个任务的进度会实时输出到终端：

```text
[视频1] 进度: 1/358 (0.28%)
[视频2] 进度: 1/376 (0.27%)
...
```

下载成功后，对应行会从 CSV 中删除，并提示：

```text
下载成功！文件保存在：./Downloads/视频1.*
已自动删除已完成的会话。
```

### 3. 查看错误日志

如果某个任务失败（返回码非 0 或未生成文件），程序会生成日志文件到 `Logs/`，文件名格式为 `YYYY-MM-DD_HH-MM-SS-ffffff.log`，内容包含完整的命令行输出、错误码等，便于排查。

## ⚙️ 配置选项

所有配置项在源代码中定义，如需修改，请重新编译：

| 常量           | 说明                   | 默认值                                   |
| -------------- | ---------------------- | ---------------------------------------- |
| `MAX_WORKERS`  | 最大并发下载数         | `5`                                      |
| `FILE_PATH`    | 任务列表 CSV 路径      | `./DownloadList.csv`                     |
| `N_M3U8DL_CLI` | 下载引擎可执行文件路径 | `./N_m3u8DL_CLI/N_m3u8DL-CLI_v3.0.2.exe` |
| `OUTPUT_DIR`   | 下载输出目录           | `./Downloads`                            |

## 💻 从源码编译

如果你希望修改代码或自行编译：

1. 安装 [Rust](https://www.rust-lang.org/)（推荐稳定版）。
2. 克隆或下载本工程源码。
3. 在项目根目录下执行：

```bash
cargo build --release
```

1. 编译后的可执行文件位于 `target/release/m3u8_downloader.exe`。

确保 `N_m3u8DL_CLI` 的路径和 `DownloadList.csv` 与可执行文件在同一级目录（或调整路径）。

------

## ⚠️ 注意事项

- 请确保 `N_m3u8DL-CLI` 有可执行权限，且支持当前操作系统。
- 下载过程中，终端输出可能因多线程交错而略微混乱，但锁机制保证最终信息完整。
- 如果 CSV 中存在空行或格式错误，程序会忽略并继续。
- 不要手动修改正在下载的任务对应的 CSV 行，以免造成文件读写冲突（程序已加锁，但仍建议等待任务完成）。
